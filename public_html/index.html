A bit about me...
I'm working on two neuroinformatics projects at McGill University...
BrainBrowser is a WebGL-based JavaScript library for 3D neurological data visualization. The source code for BrainBrowser is available on GitHub.
CBRAIN is a distributed compute platform providing connecting neuroimaging researchers to high performance computing centers across Canada and globally.
