A bit about me...
I've worked on a few major projects...
The BioDigital Human is a WebGL-based virtual 3D body that brings to life thousands of medically accurate anatomy objects and health conditions in an interactive web-based platform.
BrainBrowser is JavaScript library exposing a set of WebGL and HTML5 canvas-based tools for visualizing 3D neurological data within a web browser. The source code for BrainBrowser is available on GitHub.
CBRAIN is a distributed compute platform connecting neuroimaging researchers to distributed data stores, analysis tools and high performance computing centers across Canada and globally.

Climb is an HTML5 minimalist vertical platformer featuring music by Nadir Bellahmer. It is available to play on Newgrounds and Kongregate. Climb was built using the tgame and oFactory JavaScript libraries. The source code for Climb is available on GitHub.

Some JavaScript libraries and
applications...
nano-server is an ultra-lightweight node.js HTTP server for web development. It has no dependencies on external libraries, attemps to set the Content-type header for browser-relevant file and will send gzipped versions of files if they are available.
ScrollEm is JavaScript library for creating scrolling animations on a web page. It allows the developer to make numeric CSS properties vary based on the current vertical scroll position of the page. ScrollEm was used to build this web site.
oFactory is a JavaScript library for building factory functions. It was built as a way to avoid using constructor functions in my work. oFactory has been used in many of my games and experiments.
tgame is a simple JavaScript game engine. It implements a flexible state machine, asset loaders and helpers for basic collision detection, mouse and keyboard binding, and special effects. tgame was used to create Climb.
glUtils is a bare-bones WebGL library that abstracts away the most common boilerplate WebGL code, while still allowing one to use the API directly. glUtils was used to build my WebGL experiments.
